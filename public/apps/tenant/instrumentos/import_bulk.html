Importar instrumentos desde archivo

Cómo funciona

  • Descarga la plantilla oficial para respetar el orden de las columnas.
  • Completa los datos obligatorios: catálogo, marca, modelo, serie, código, departamento, ubicación y fecha de alta.
  • Guarda el archivo en formato CSV o Excel (.xlsx) y súbelo desde esta pantalla.
  • El sistema validará cada fila, normalizará los catálogos y derivará el estado del instrumento automáticamente.
